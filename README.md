## Prerequisites

- Windows with PowerShell
- [Scoop](https://scoop.sh/)

## How to use

**Add this bucket**

```powershell
scoop bucket add arturo https://github.com/arturo-lang/scoop-bucket
```

**Installing**

```powershell
scoop install arturo
```


## Update and remove

```powershell
# Refresh manifests and installed apps
scoop update

# Remove Arturo
scoop uninstall arturo

# Remove the bucket if you no longer need it
scoop bucket rm arturo
```
